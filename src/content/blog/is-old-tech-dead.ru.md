Сейчас в поиске работы изучаю проекты потенциальных работодателей. И раз за разом нахожу в них то, что в сообществе давно считается устаревшим. Между тем что в тренде и тем что реально в продакшне - пропасть.

Каждый год выходит пост "jQuery мёртв". Но jQuery об этом не знает, в 2026-м он на 77.8% сайтов по данным W3Techs.

PHP умирает с 2012, между тем WordPress работает на 43% сайтов интернета, а Laravel продолжает получать мажорные релизы и при этом неплохо дружит с React/Vue через Inertia.js.

У React убийц тоже хватало: Svelte, Solid, Qwik. Каждый обещал его заменить. React при этом используют 83% разработчиков по данным State of JS опроса за 2025 год.

GraphQL вышел в 2015 с обещанием переписать подход к API целиком. Стал популярным, но REST никуда не делся: GraphQL нашёл свою нишу рядом, а не вместо.

Deno вышел в 2020: нативный TypeScript, встроенный fetch, без npm, "наконец всё сделано правильно". Bun в 2022 пришёл с бенчмарками, где он в разы обходит Node по скорости. В итоге Node.js всё равно используют 90% разработчиков. При этом в Node 22 появился нативный TypeScript, встроенный fetch появился ещё в Node 18.

Похожая история с npm: его "исправляли" Yarn в 2016, pnpm в 2017, Bun пришёл со своим менеджером. npm идёт в комплекте с Node, и большинству этого оказывается достаточно.
